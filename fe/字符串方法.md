# 常用的字符串方法

```
var s = 'hello World'

s.length     //字符串长度
11

s.charAt(0)   //第一个字符
"h"

s.charAt(s.length - 1)    //最后一个字符
"d"


s.substring(1,5)	//取子集，下标1到5的字符
"ello"

s.slice(1,5)	//取子集，下标1到5的字符
"ello"
s.slice(1)		//如果只有一个正数参数，默认第二个参数是字符串最后一位
ello World

s.slice(-1)		//如果只有一个参数且是负数，取最后的几位

s.indexOf('r')		//某个字符第一次出现的位置,
				1.如果该字符串有两个这样的字符，返回第一个的位置。
				2.接受多个字符
				3.如果输出不存在的字符,返回-1，无参数也返回-1
s.indexOf('r',3)	//3位置之后r第一次出现的位置
				
s.lastIndexOf('r');		//最后一次出现的位置

s.split(" ");		//以某一个字符为分割点，分割成子字符串，返回子字符串的数组。
["hello", "World"]

s.replace("r","22")		//将某个字符替换成其他字符,原理是正则。

```

### 注意

1.字符串方法都不修改字符串本身，返回值是一个新的字符串。
2.字符串是一种数组，可以使用下标访问s[0].

```
s[0]
h
```
